What You Get

Everything in one self-contained Python app

Real Terminal Emulator

Full xterm.js terminal with PTY backend. Tmux session persistence survives page reloads and reconnects. Multiple tabs.

Chat Interface

Persistent conversations with streaming responses, markdown rendering, syntax highlighting, and session export.

SSH Remote Servers

5-step wizard: auto-generate SSH keys, push to remote, verify connection. Import from ~/.ssh/config. Nobody else has this.

Multi-User Auth + RBAC

Password auth with bcrypt, admin/user roles, rate limiting. Scoped terminal output per user. 191 security tests.

Autonomous Mode

Set a task, walk away. Claude works in tmux with auto-restart. Monitor progress from your phone.

PWA + Mobile

Install as a native app on your phone or tablet. Offline-capable service worker. Full functionality on any screen size.
